附录B 消防车活荷载考虑覆土厚度影响的折减系数

B.0.1 当考虑覆土对楼面消防车活荷载的影响时，可对楼面消防车活荷载标准值进行折减，折减系数可按表B.0.1、表B.0.2采用。

表B.0.1 单向板楼盖楼面消防车活荷载折减系数
折算覆土厚度s（m） 楼板跨度（m）
2 3 4
0 1.00 1.00 1.00
0.5 0.94 0.94 0.94
1.0 0.88 0.88 0.88
1.5 0.82 0.80 0.81
2.0 0.70 0.70 0.71
2.5 0.56 0.60 0.62
3.0 0.41 0.51 0.54

表B.0.2 双向板楼盖楼面消防车活荷载折减系数
折算覆土厚度s（m） 楼板跨度（m）
3×3 4×4 5×5 6×6
0 1.00 1.00 1.00 1.00
0.5 0.95 0.96 0.99 1.00
1.0 0.88 0.93 0.98 1.00
1.5 0.79 0.83 0.93 1.00
2.0 0.67 0.72 0.81 0.92
2.5 0.57 0.62 0.70 0.81
3.0 0.48 0.54 0.61 0.71

B.0.2 板顶折算覆土厚度s应按下式计算：

$$\bar{s} =1.43s\tan \theta \tag{B.0.2}$$

式中 s —— 覆土厚度（m）；
θ —— 覆土应力扩散角，不大于45°。
