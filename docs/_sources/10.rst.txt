10 偶然荷载
===============



.. raw:: html

  <h2 id="test111">10 偶然荷载</h2>

10.1 一般规定
---------------------------------
.. raw:: html

 <p style="text-align:justify;"><a href="#id10.1.1">10.1.1</a> <span id="id10.1.1">偶然荷载应包括爆炸、撞击、火灾及其他偶然出现的灾害引起的荷载。本章规定仅适用于爆炸和撞击荷载。</span></p>
 <p style="text-align:justify;"><a href="#id10.1.2">10.1.2</a> <span id="id10.1.2">当采用偶然荷载作为结构设计的主导荷载时，在允许结构出现局部构件破坏的情况下，应保证结构不致因偶然荷载引起连续倒塌。</span></p>
 <p style="text-align:justify;"><a href="#id10.1.3">10.1.3</a> <span id="id10.1.3">偶然荷载的荷载设计值可直接取用按本章规定的方法确定的偶然荷载标准值。</span></p>

10.2 爆炸
---------------------------------

.. raw:: html

 <p style="text-align:justify;"><a href="#id10.2.1">10.2.1</a> <span id="id10.2.1">由炸药、燃气、粉尘等引起的爆炸荷载宜按等效静力荷载采用。</span></p>
 <p style="text-align:justify;"><a href="#id10.2.2">10.2.2</a> <span id="id10.2.2">在常规炸药爆炸动荷载作用下，结构构件的等效均布静力荷载标准值，可按下式计算：</span></p>

$$q_{ce}=K_{dc}P_{c}\\tag{10.2.2}$$

 <p style="text-align:justify;"><a href="#id10.2.3">10.2.3</a> <span id="id10.2.3">对于具有通口板的房屋结构，当通口板面积<i>A</i><sub>V</sub>与爆炸空间体积V之比在0.05~0.15之间且体积V小于1000m³时，燃气爆炸的等效均布静力荷载<i>P</i><sub>k</sub>可按下列公式计算并取其较大值：</span></p>

$$P_{k}=3+P_{v}\\tag{10.2.3-1}$$

$$P_{k}=3+0.5P_{v}+0.04\\left ( \\frac{A_{V}}{V}  \\right ) ^{2} \\tag{10.2.3-2}$$

10.3 撞击
---------------------------------

.. raw:: html

 <p style="text-align:justify;"><a href="#id10.3.1">10.3.1</a> <span id="id10.3.1">电梯竖向撞击荷载标准值可在电梯总重力荷载的（4~6）倍范围内选取。</span></p>
 <p style="text-align:justify;"><a href="#id10.3.2">10.3.2</a> <span id="id10.3.2">汽车的撞击荷载可按下列规定采用：</span></p>
 <ol>
 <li>顺行方向的汽车撞击力标准值<i>P</i><sub>k</sub>（kN）可按下式计算：</li>
 </ol>

$$P_{k}=\\frac{mv}{t} \\tag{10.3.2}$$

.. raw:: html
 
 <table border="0" style="font-family:times new roman" id="gongshi">
 <tr>
 <td width="50px" align='center' id="eqzs">式中</td>
 <td width="30px" align='left' id="eqzs">m</td>
 <td width="40px" align='left' id="eqzs">——</td>
 <td id="eqzs">汽车质量（t），包括车自重和载重；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs">v</td>
 <td id="eqzs">——</td>
 <td id="eqzs">车速（m/s）；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs">t</td>
 <td id="eqzs">——</td>
 <td id="eqzs">撞击时间（s）。</td>
 </tr>
 </table>
 <p></p>
 <ol start="2">
 <li style="text-align:justify;">撞击力计算参数m、v、t和荷载作用点位置宜按照实际情况采用；当无数据时，汽车质量可取15t，车速可取22.2m/s，撞击时间可取1.0s，小型车和大型车的撞击力荷载作用点位置可分别取位于路面以上0.5m和1.5m处。</li>
 <li style="text-align:justify;">垂直行车方向的撞击力标准值可取顺行方向撞击力标准值的0.5倍，二者可不考虑同时作用。</li>
 </ol>
 <p style="text-align:justify;"><a href="#id10.3.3">10.3.3</a> <span id="id10.3.3">直升飞机非正常着陆的撞击荷载可按下列规定采用：</span></p>
 <ol>
 <li>竖向等效静力撞击力标准值<i>P</i><sub>k</sub>（kN）可按下式计算：</li>
 </ol>

$$P_{k}=C\\sqrt{m}  \\tag{10.3.2}$$

.. raw:: html
 
 <table border="0" style="font-family:times new roman" id="gongshi">
 <tr>

 <td width="30px" align='left' id="eqzs">C</td>
 <td width="40px" align='left' id="eqzs">——</td>
 <td id="eqzs">系数，取3kN·<i>kg</i><sup>-0.5</sup>；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs">m</td>
 <td id="eqzs">——</td>
 <td id="eqzs">直升飞机的质量（kg）。</td>
 </tr>
 </table>
 <p></p>
 <ol start="2">
 <li style="text-align:justify;">竖向撞击力的作用范围宜包括停机坪内任何区域以及停机坪边缘线7m之内的屋顶结构。</li>
 <li style="text-align:justify;">竖向撞击力的作用区域宜取2mx2m。</li>
 </ol>
